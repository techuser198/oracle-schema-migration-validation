# Installation Notes:
# ==================
# 1. Install system packages first:
#    Ubuntu/Debian: sudo apt update && sudo apt install -y bash diffutils python3-pip
#    RHEL/CentOS: sudo yum install -y bash diffutils python3-pip
#    
# 2. Install Oracle Instant Client:
#    - Download from Oracle website
#    - Or use package manager if available
#    - Ensure sqlplus is in PATH
#
# 3. Install Python dependencies:
#    pip install -r requirements.txt
#
# 4. Install Ansible collections:
#    ansible-galaxy collection install ansible.posix community.general
#
# 5. Verify installation:
#    ansible --version
#    sqlplus -v
#
# Schema Validation Features:
# ==========================
# The validate-schema operation compares 11 different Oracle object types:
# - Tables, Columns, Sequences, Indexes, Constraints
# - Triggers, Views, Synonyms, Privileges
# - Object counts and database structure
# 
# Uses Jinja2 templates for dynamic script generation and supports
# automatic cleanup of old comparison files.
